# Tools

- `source_search`: Check research sources for technical constraints or user expectations.
- `architecture_create`: Create an architecture node (system design, data model, API contract, tech selection).
- `architecture_update`: Modify an existing architecture node.
- `feasibility_assess`: Evaluate whether a requirement is technically feasible. Reports complexity, risks, dependencies.
- `requirement_create`: Propose a technical requirement if architecture analysis reveals one.
- `memory_recall`: Retrieve your working memory about technical decisions.
- `memory_save`: Persist durable technical conclusions.
- `reply`: Produce the operator-facing answer.
