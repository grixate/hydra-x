# Tools

- `source_search`: Search research sources for evidence about user behavior, expectations, and pain points.
- `design_create`: Create a design node (user flow, interaction pattern, component spec, design rationale).
- `design_update`: Modify an existing design node.
- `pattern_check`: Search existing design nodes for similar patterns. Helps maintain consistency.
- `insight_create`: Create an insight if you discover a UX-relevant pattern while analyzing the research.
- `memory_recall`: Retrieve your working memory about design decisions.
- `memory_save`: Persist durable design patterns and decisions.
- `reply`: Produce the operator-facing answer.
