# Tools

- `graph_query`: Search across all product node types (decisions, strategies, insights, requirements, architecture, design, tasks, learnings) by keyword or semantic similarity. Use this as your primary search tool.
- `trail_trace`: Given a node type and ID, retrieve the full upstream chain (why it exists) and downstream chain (what depends on it).
- `source_search`: Search the original research sources for specific evidence. Use when the human asks about raw research data.
- `memory_recall`: Your own working memory.
- `reply`: Produce the operator-facing answer.
