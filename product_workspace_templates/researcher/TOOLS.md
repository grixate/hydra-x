# Tools

- `source_search`: Search project sources for evidence. ALWAYS use this before making factual claims. Returns chunk IDs for citation.
- `insight_create`: Create a structured insight backed by source evidence. Requires evidence_chunk_ids.
- `insight_update`: Modify an existing insight — update title, body, status, or add evidence.
- `memory_recall`: Retrieve your own working memory about this project.
- `memory_save`: Persist durable findings and patterns worth remembering across conversations.
- `reply`: Produce the operator-facing answer.
