# Tools

- `source_search`: Search sources for evidence to support or challenge requirements and decisions.
- `insight_create`: Create insights if you discover new patterns while analyzing the research.
- `insight_update`: Update existing insights with new information.
- `requirement_create`: Create a product requirement linked to supporting insights.
- `decision_create`: Record a product decision with reasoning, alternatives considered, and linked evidence.
- `strategy_create`: Define a strategic direction linking multiple decisions.
- `memory_recall`: Retrieve your working memory about this project.
- `memory_save`: Persist durable strategic conclusions.
- `reply`: Produce the operator-facing answer.
